Melbourne Day Tours

I mocked up and coded this landing page without the use of a CMS and pre-determined style sheets. The concept was to create a landing page purely dedicated to promote and sell Day Tours without the distractions of the other type of AAT Kings products. Instead of focusing on tour names and descriptions, I chose to hone in on the highlights and destinations of where a Day Tour might go and categorise them accordingly.

The page opens with captivating full-length full-width banners and paralax scrolling on an tour grid. There are hover states over each grid image and smooth scrolling to tours when clicked. Tour discounts were always overlooked so I incorporated those in an interactive fashion; and also showed the simplicity of how to book and join a tour with interactive sliding boxes and hotel pick up lists. I also included a sticky bar and navigation to sight wheelchair and fitness accessibility to really focus on breaking down the walls that a guest may have when looking for a day tour. Finally there is a cross sell/up sell to the Short Break product which directly relates to the advertised Day Tours of the page.

AAT Kings eDM + Exact Target Template

Back in the day I used to get a HTML eDM brief from the marketing coordinator to build; and after multiple edits and back and forth, use to hand over the HTML code for them to distribute as needed.

These days, the process is much simpler after I created several resposive eDM templates in Exact Target (Sales Force/Marketing Cloud). After much trial and error, I understood how Exact Target changed the code from input to output, and created the templates with the non-html user in mind. I developed them in a way that content and images could be dropped in with minimal html knowledge, and created a Content library with pre-determined styling (eg headings, body text, buttons, promo blocks etc).

Today I edit a template every few months usually to alter a layout, thus leaving more time for website enhancements and other projects.

Trade Website Manual

This guide was put together for Travel Agents using the new AAT Kings Trade Website. The manual is a step by step guide on how to register a travel agency, and how to troubleshoot if they are not able to. It offers navigation around the Trade Website and instructions such as 'How to order Brochures' or 'Using our Resources'. The Manual has similar brand aesthetic and layout as the actual Trade Portal for ease of use.

The Trade Website Manual is built with raw html code and without any CMS assistance or pre-determined style sheets. As the prime user (travel agents) are usually desktop-based, the manual is not responsive and is one of my earlier html projects I've created.
